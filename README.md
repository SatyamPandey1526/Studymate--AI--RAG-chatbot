# StudyMate AI — Streamlit RAG Chatbot (Groq + Chroma + e5 embeddings)

![StudyMate AI](https://images.unsplash.com/photo-1522202176988-66273c2fd55f?q=80\&w=1200\&auto=format\&fit=crop\&ixlib=rb-4.0.3)

---

## **Overview**

**StudyMate AI** is an advanced Retrieval-Augmented Generation (**RAG**) chatbot built with **Streamlit**, **Groq LLM**, **ChromaDB**, and **e5 embeddings**. It allows you to:

* Upload study materials (**PDF**, **DOCX**, **TXT**)
* Index and persist them using **ChromaDB**
* Ask questions and receive **context-aware answers**
* Keep **multiple chat conversations**
* View and download **source documents**
* Optionally enable a **reranker** for better retrieval accuracy
* Automatically **detect and skip sensitive information**

---

## **Features** ✨

### **Document Management**

* Upload multiple **PDF**, **DOCX**, and **TXT** files
* Automatically chunks and embeds documents
* Persistent **ChromaDB** storage — documents remain indexed between sessions

### **Chat Capabilities**

* Ask natural language questions about your uploaded materials
* Context-aware answers generated by **Groq LLM**
* Displays **sources** with **download buttons** for transparency

### **Conversation Management**

* Create and manage multiple conversations
* Auto-title conversations based on the first question

### **Reranking (Optional)**

* Uses **cross-encoder/ms-marco-MiniLM-L-6-v2**
* Improves retrieval accuracy by reranking relevant chunks

### **Sensitive Data Protection**

* Detects and skips chunks containing:

  * Passwords, API keys, tokens
  * Financial info (credit card, bank account)
  * Aadhaar, PAN, SSN numbers
  * Emails and other confidential data

| Component             | Technology                                                 |
| --------------------- | ---------------------------------------------------------- |

| **Env Management**    | [python-dotenv](https://pypi.org/project/python-dotenv/)   |

## **Setup Instructions** 🚀

### **1. Clone the Repository**

```bash
git clone https://github.com/sayprincekumar20/studymate_bot_update.git
cd studymate_bot_update
```

### **2. Create and Activate Virtual Environment**

```bash
python -m venv venv
source venv/bin/activate   # On Windows: venv\Scripts\activate
```

### **3. Install Dependencies**

```bash
pip install -r requirements.txt
```

### **4. Set Up Environment Variables**

Create a `.env` file in the project root:

```ini
GROQ_API_KEY=your_groq_api_key_here
```

> ⚠️ **Important**: `.env` is ignored via `.gitignore` to prevent leaking secrets.

### **5. Run the App**

```bash
streamlit run app.py
```

## **Usage Guide** 📖

1. **Upload Documents** via the sidebar.
2. Click **Add to Knowledge Base** to embed and index them.
3. Ask natural language questions about your documents.
4. **Download sources** of the retrieved context if needed.
5. Use **"New Conversation"** for different topics.
